Google Drive
  1. Enter credentials and click Save Config.
  2. Connect each bank — click Connect, complete the bank login in the popup.
  3. Fetch transactions — set a date range and click Fetch All Banks.
  4. Download as CSV or save to Google Drive.
Google Drive requires a GCP project with Drive API enabled and OAuth 2.0 credentials.
  1. Go to console.cloud.google.com and enable the Google Drive API
  2. Create OAuth 2.0 Client ID (Web application) with your Railway URL as authorised origin and redirect URI
  3. Paste the Client ID in Config above
  4. Click Connect Google Drive below
