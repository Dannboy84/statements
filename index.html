How It Works
Google Drive
  1. Create a GoCardless account at bankaccountdata.gocardless.com and get a Secret ID + Secret Key from the developer portal.
  2. Connect each bank — Click "Connect" on each bank card. You will be redirected to GoCardless's secure auth flow.
  3. Fetch statements — After connecting, choose a date range and click Fetch.
  4. Download or save to Drive — Export as CSV or push to Google Drive.
Google Drive integration requires a GCP project with the Drive API enabled and OAuth 2.0 credentials set up.
  1. Go to console.cloud.google.com → APIs & Services → Enable "Google Drive API"
  2. Create OAuth 2.0 Client ID → Web application → add your Railway URL as authorised origin and redirect URI
  3. Paste your Client ID in the Config tab above
  4. Click "Connect Google Drive" below to authorise.
