How It Works
Google Drive
  1. Enter credentials and click Save Config.
  2. Connect each bank — click Connect, complete the bank login in the popup.
  3. Fetch transactions — set a date range and click Fetch All Banks.
  4. Download as CSV or save to Google Drive.
Google Sheets integration — data is written directly into your "LunchMoney Start" spreadsheet. Nationwide appends to the bottom of Nationwide - BS. Tesco inserts new rows after the header in Tesco - BS.
  1. Go to console.cloud.google.com and enable both Google Drive API and Google Sheets API
  2. Create OAuth 2.0 Client ID (Web application) with your Railway URL as authorised origin and redirect URI
  3. Add your Google account as a test user under OAuth consent screen
  4. Paste the Client ID in Config above, then click Connect Google Drive below
